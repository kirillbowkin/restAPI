RESTful API is online

You can try it out using "Postman" or your own API client


List of commands


(GET) /api/v1/customers - to get all customers

(GET) /api/v1/customers/{id} - to get specific customers

(POST) /api/v1/customers/ - to add customers

(PUT) /api/v1/customers/ - to update customer(you'll need to specify customer's id in the very first line of JSON file, eg. "id" : "1")

(DELETE) /api/v1/customers/{id} - to delete specific customers
